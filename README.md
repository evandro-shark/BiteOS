# BiteOS
BiteOS – Sistema Operacional Completo para Lanchonetes (Multilanguage: Português Brasil + English) Descrição Geral Crie um ecossistema completo chamado BiteOS, composto por 6 aplicativos integrados, todos compartilhando a mesma base de dados. O sistema deve ser multilíngue, permitindo alternância de idioma em qualquer módulo.
